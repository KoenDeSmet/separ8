Gmail setup
Separ8 gebruikt de Anthropic API met je Gmail connector om het label separ8 te lezen.
1
Gmail filter aanmaken (eenmalig)
Van: vinted.be OR 2dehands.be OR facebookmail.com OR ebay.be
→ Label toepassen: separ8
2
Gmail ID instellen
Vul hieronder je Gmail-label ID in (zie je inbox instellingen). Standaard werkt separ8 als zoekopdracht.
3
Synchroniseer
Tab Inbox → klik Synchroniseer → berichten verschijnen automatisch.
